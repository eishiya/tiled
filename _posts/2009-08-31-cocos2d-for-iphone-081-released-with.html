Its most recent release adds support for Tiled by directly supporting the TMX map format. A guide is available describing how to use cocos2d and Tiled. Thanks, Ricardo!

This is not the first time a 3rd party decides to add support for Tiled. Our wiki lists some other projects, like Kevin Gadd's XNA map loader, dr0id's Pygame map loader and Slick2D, a Java game library based on LWJGL with Tiled support. Thanks!
